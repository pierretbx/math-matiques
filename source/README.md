# Organisation des ressources

Les fichiers de `source/a trier` ont été rangés dans les dossiers suivants :

- `seconde/` : chapitres, exercices, DS, polycopiés, questions flash et automatismes de Seconde.
- `premiere/technologique/` : documents explicitement identifiés comme Première technologique.
- `premiere/specialite/` : dossiers de Première spécialité et contenus de Première que tu as indiqués comme spécialité.
- `premiere/specifique/` : dossier prêt, vide pour l'instant.
- `terminale/complementaires/` : documents de mathématiques complémentaires.
- `terminale/specialite/` : chapitres principaux de Terminale spécialité.
- `terminale/expertes/` : complexes, arithmétique, matrices, graphes et approfondissements.
- `terminale/a-classer/` : documents ambigus, technologiques ou divers à valider plus tard.

Le dossier `a trier/` ne contient normalement plus que des fichiers système ou temporaires ignorés.
